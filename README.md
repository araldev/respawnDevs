# 🍽️ Proyecto Restaurante - Gestión de Platos y Alérgenos

## 📌 Descripción del proyecto

Este proyecto es una aplicación web desarrollada con **Angular (frontend)** y **Spring Boot (backend)** que permite visualizar y gestionar información de distintos platos de un restaurante.

La aplicación está enfocada en la **presentación clara de información alimentaria**, especialmente en lo relacionado con **alérgenos**, para ayudar a los usuarios a tomar decisiones informadas sobre lo que consumen.

Cada plato mostrado en la aplicación incluye información detallada como:
- 📷 Imagen del plato
- 📝 Descripción del plato
- 💶 Precio
- ⚠️ Iconos de alérgenos asociados
- 🍽️ Nombre del plato

Toda esta información está conectada a una **base de datos**, lo que permite que los datos sean dinámicos y gestionados desde el backend.

---

## 🧠 Objetivo del proyecto

El objetivo principal es crear una plataforma moderna, intuitiva y visualmente atractiva que permita:

- Mostrar platos de un restaurante de forma clara
- Informar sobre alérgenos de cada alimento
- Mejorar la experiencia del usuario en la elección de comida
- Conectar frontend y backend mediante API REST
- Practicar el desarrollo full stack con Angular + Spring Boot

## 🧩 Funcionalidades principales

- 📋 Visualización de tarjetas de platos
- 🖼️ Imágenes representativas de cada plato
- ⚠️ Sistema de iconos de alérgenos (gluten, lactosa, frutos secos, etc.)
- 💰 Visualización del precio de cada plato
- 📝 Descripciones detalladas de los platos
- 🔗 Conexión con base de datos mediante API REST
- 🔄 Datos dinámicos (cargados desde backend)
